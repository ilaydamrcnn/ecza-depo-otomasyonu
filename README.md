# Ecza Depo Otomasyonu

C# ile geliştirilmiş **eczane depo otomasyonu ve stok yönetim sistemi**.  
Bu uygulama, eczanelerde depo yönetimini kolaylaştırmak, ilaç stoklarını takip etmek ve günlük işlemleri otomatikleştirmek amacıyla tasarlanmıştır.

## Özellikler
- Depo stok takibi
- Yeni ürün ekleme ve güncelleme
- Ürün silme
- Stok raporları görüntüleme
- Kullanıcı dostu masaüstü arayüz
